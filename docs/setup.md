# Setup Guide

This guide will help you set up the repository locally and run the LeetCode solutions.

## Prerequisites

Before you start, make sure you have the following installed:

### Required
- **Java Development Kit (JDK) 8 or higher**
  - [Download JDK](https://www.oracle.com/java/technologies/downloads/)
  - Verify installation: `java -version`
  
### Optional (for build automation)
- **Maven 3.6+** (for building and testing)
  - [Download Maven](https://maven.apache.org/download.cgi)
  - Verify installation: `mvn --version`

- **Gradle 7.0+** (alternative to Maven)
  - [Download Gradle](https://gradle.org/install/)
  - Verify installation: `gradle --version`

- **IDE** (optional but recommended)
  - [IntelliJ IDEA](https://www.jetbrains.com/idea/)
  - [Eclipse](https://www.eclipse.org/)
  - [VS Code with Java Extension](https://code.visualstudio.com/)
