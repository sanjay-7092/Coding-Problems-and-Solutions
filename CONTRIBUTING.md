# Contributing Guidelines

Thank you for your interest in contributing! Here's how to add new solutions:

## Steps to Add a Solution

1. **Create a folder** with the problem number and name:
   ```
   problems/[difficulty]/problem-[id]-[name]/
   ```

2. **Create three files:**
   - `problem-description.md` - Problem statement
   - `[ProblemName].java` - Solution code
   - `test-cases.md` - Test cases

3. **Follow the code template:**
   - Include class javadoc with problem ID and difficulty
   - Add method javadoc with complexity analysis
   - Use descriptive variable names
   - Add comments for complex logic

4. **Submit a Pull Request** with:
   - Clear description of the solution approach
   - Complexity analysis
   - Any edge cases handled

## Code Style
- Follow Google Java Style Guide
- Use meaningful variable names
- Include complexity analysis in comments
- Add example test cases
